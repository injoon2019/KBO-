## 분석하고 싶은 것:
  1. KBO 연봉 분포, 연봉에 따른 실력 차이가 존재하는가, 고액 연봉을 받지만 성적을 못내는 먹튀는 누구인가? (인준)
  2. 구단별 비만도 조사 및 경기력에 미치는 영향  (시몬)
  3. 팀별 투수, 타자, 수비 각각의 성적이 구단성적에 얼마나 영향을 끼치는가? (범희)
  4. 타자의 키/체중과 도루 등의 연관성은?  (동현)
